Bystfy

A Chunk website. An app that enables users split between CSV or JSON files.

This on-going responsive project aims at enabling users split files in CSV or JSON format, these files can be saved to users account or downloaded to users computer.

Roles

UX Writing

User Research

Visual Design

Tools

Miro

Duration

July-August 2022

Thesis@ZuriTraining

Problem Statement

Communication

Over the years, resizing files and having them maintain the same quality while functional can be worrisome for a data analyst, receptionist, student, or in any profession especially when it affects productivity.

Digitilization

Users will always settle for what they have or not delve into in completely when they have limited or no options. Users needed a platform to quickly upload their files and have a functional file in smaller bits to work with.

Bytsfy

CSV or JSON files can be resized into bits while maintaining the same quality as well as an option to download to computer and save for later.

  • How might we enable CSV and JSON format only?
  • How might we have users save files after split, compress or convert?
  • How might we ensure users maintain original quality after chunking?

What users see?

Unautheticated user

  • View basic information
  • Access to documentation (How Btysfy works)
  • Unable to chunk files

Autheticated user

  • Full access to the platform
  • Access to split files by lines
  • Access to choose output format i.e CSV or JSON
  • Allows for save and download to computer.
